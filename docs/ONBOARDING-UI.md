## âœ… Validation Rules

### License Key
- Pattern: `[A-Z]{3}-[A-Z0-9]{5}-[A-Z0-9]{5}-[A-Z0-9]{4}-[A-Z0-9]{4}`
- Example: `PRM-4XY9Z-8K2L4-M7N3P-C4F6`

### Email
- Pattern: RFC 5322 compliant
- Example: `user@example.com`

### Mobile Number
- Pattern: E.164 format
- Example: `+919876543210`

### Password
- Minimum 8 characters
- At least 1 uppercase letter
- At least 1 lowercase letter
- At least 1 number
- At least 1 special character

### GST Number (India)
- Pattern: `[0-9]{2}[A-Z]{5}[0-9]{4}[A-Z]{1}[1-9A-Z]{1}Z[0-9A-Z]{1}`
- Example: `33AAACR7749E2ZD`

### PAN Number (India)
- Pattern: `[A-Z]{5}[0-9]{4}[A-Z]{1}`
- Example: `AAACR7749E`

## ğŸ¯ Best Practices
